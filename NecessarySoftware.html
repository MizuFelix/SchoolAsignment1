Als erstes brauch man einen Weg die Grundlage seines Spieles

zu erstellen. Entweder schreibt man den kompletten Code seines

Spieles selbst, oder benutzt ein externes Werkzeug, beispielsweiswe

eine sogenannte "Game Engine".


Bsp.: Unreal Engine


Mit dem Werkzeug "Unreal Engine" wurde beispielsweise

das Spiel Fortnite erstellt.


Nun, wenn man sich entschieden hat ein externes Werkzeug zu

benutzen oder sein komplettes Spiel selbst zu schreiben,

brauch man einen Weg seinen Code zu schreiben.

Empfohlen ist Microsofts "Visual Studio" oder "Visual Studio,

welches eher für die Sprachen C#, C und C++ benutzt wird.
